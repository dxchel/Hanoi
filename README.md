# Hanoi

## Description

This program was made in 2021 to learn ncurses, before using git on it

Hanoi towers game using NCurses and trying not to use many advanced C++ features like vectors
It has an interactive menu, the user can modify the number of disks
It has restrictions using the window size

The user can play the game or watch the program solve the problem in an animated way with the minimum amount of moves
